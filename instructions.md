# Monthly Reporting Task (EDIT ME)

When asked to "execute for last month":

1. Read memory.md first - apply every rule in it
2. Auto-detect the last full calendar month
3. setSessionDefaults with the company + report suite from memory.md
4. Run these reports (default segment applied to every one):
   - Daily unique visitors (variables/daterangeday)
   - Platform split (per the platform rule in memory.md)
   - Top 10 markets (per the market rule in memory.md)
   - Top 10 pages
   - Same KPIs for the previous month (for month-on-month comparison)
5. Write a 3-bullet executive summary - plain business English, no jargon
6. Build a PowerPoint using the .potx template and design_rules.md in this folder:
   - Slide 1: Title
   - Slide 2: Executive summary
   - Slide 3: KPIs with MoM change arrows
   - Slide 4: Platform split (pie)
   - Slide 5: Top markets (bar)
   - Slide 6: Top pages (bar)
   - Slide 7: Recommended next actions (5 bullets)
7. Save as <Company>_Analytics_<Month><Year>.pptx to the Desktop
8. Do not ask for approval between steps - run end to end, then summarise what you did
