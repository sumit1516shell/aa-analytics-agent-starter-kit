HOW TO SCHEDULE THE MONTHLY RUN
================================

MAC - using cron
----------------
1. Open Terminal
2. Type:  crontab -e
3. Add this line (runs 8am on the 1st of every month):

   0 8 1 * * cd ~/Desktop/aa-analytics-agent-starter-kit && python3 get_token.py > .token && copilot --agent "Read instructions.md and execute for last month"

4. Save and close. Done.

WINDOWS - using Task Scheduler
------------------------------
1. Press Start → type "Task Scheduler" → open it
2. Click "Create Basic Task"
3. Name: Monthly Analytics Deck
4. Trigger: Monthly → day 1 → 8:00 AM
5. Action: Start a program → point it at run_monthly.bat (below)
6. Finish

Create run_monthly.bat in this folder containing:

   cd %USERPROFILE%\Desktop\aa-analytics-agent-starter-kit
   python get_token.py > .token
   copilot --agent "Read instructions.md and execute for last month"

NOTE: the exact Copilot CLI command depends on your installed version -
run "copilot --help" to check yours. If you don't use the CLI, simply
open VS Code on the 1st of the month and type the one-line prompt instead.
